UserDataManager:

Android application built with:
- Kotlin
- MVVM Architecture
- Room Database
- Hilt Dependency Injection
- Material Design 3
- RecyclerView
- ViewBinding

Features
- Add User
- Edit User
- Delete User
- single selection and multiple selection delete
- search user by name or phone number
- Local Storage with Room
- Input Validation
